Motor Option

12V DC Motor

The 12V DC electronic motor is a low-voltage motorised control option designed for SteelZip outdoor blind systems. Instead of being connected directly to 240V mains power, this motor operates using a 12V DC battery pack, making it a practical option where hardwiring is not preferred or where a simpler power setup is required. It provides smooth remote-control operation and includes obstacle detection for added safety and convenience.

  • Powered by 12V DC battery pack
  • No direct 240V hardwiring required
  • Electronic motor with obstacle detection
  • Remote control operation
