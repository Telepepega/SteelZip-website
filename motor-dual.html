Motor Option

DUAL CONTROL Motor

The dual control motor combines electric remote operation with manual hand control in one system. It allows the blind to be operated normally by remote control, while still giving the user the ability to manually operate the blind when required, including during power outages. With obstacle detection included, the system also helps protect the blind during operation.

  • Electric remote control operation
  • Manual, remote or control options available
  • Obstacle detection for added protection
  • Practical during power outages
